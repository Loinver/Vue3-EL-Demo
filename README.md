# 一、项目简介

```
一个基于Vue3快速生成PC端Vue代码的模板项目
```

# 二、快速上手

## 2.1 安装依赖

推荐使用 yarn 管理 npm 依赖

```
yarn install
```

## 2.2 本地开发

```
yarn serve
```

## 2.3 提交校验

```
yarn commit
```

## 2.4 构建部署

```
yarn build
```
